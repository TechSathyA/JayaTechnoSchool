# JayaTechnoSchool

Static multi-page demo website for **Jaya Techno School - Hassan**.

## Overview

This repository contains a presentation-ready school website built with plain HTML, CSS, and JavaScript. It is designed for school profile sharing, admissions-focused communication, and future migration into tools like WordPress or Elementor.

## Live Site

- GitHub Pages URL: `https://techsathya.github.io/JayaTechnoSchool/`

## Project Structure

- `jaya-techno-school-demo/` - website source files
- `jaya-techno-school-demo/assets/images/` - placeholder graphics
- `jaya-techno-school-demo/css/style.css` - global site styling
- `jaya-techno-school-demo/js/script.js` - navigation and form behavior
- `jaya-techno-school-demo.zip` - packaged copy of the demo site

## Included Pages

- Home
- About
- Academics
- Techno Learning
- Facilities
- Admissions
- Gallery
- Contact

## Tech Stack

- Plain HTML
- Plain CSS
- Minimal vanilla JavaScript

## Local Preview

Run the site locally from the project folder:

```powershell
cd jaya-techno-school-demo
python -m http.server 8000
```

Then open:

- `http://127.0.0.1:8000/`

## Publishing

GitHub Pages deployment is configured through GitHub Actions and publishes the contents of `jaya-techno-school-demo/`.

## Notes

- Built as a presentation/demo website.
- Includes the required label: **Demo Website - For Presentation Only**.
- Uses placeholder visuals marked **Replace with actual school images**.
